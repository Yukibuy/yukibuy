Guide export

Comment exporter votre fichier CSV ?

Simple et rapide. Environ 3 minutes. Un guide détaillé est envoyé après paiement.

Google Ads

  1. Connectez-vous à votre compte Google Ads
  2. Menu "Campagnes" → sélectionnez toutes les campagnes
  3. Période : 30 derniers jours
  4. Bouton "Télécharger" → Format CSV
  5. Colonnes utiles : Campagne, Impressions, Clics, Coût, Conversions, Valeur des conversions

Meta Ads

  1. Ouvrez le Gestionnaire de publicités Meta
  2. Vue "Campagnes" → sélectionnez toutes
  3. Période : 30 derniers jours
  4. Bouton "Exporter" → Exporter les données de tableau (CSV)
  5. Colonnes utiles : Campagne, Dépense, Clics, Achats, Valeur d'achat, CPA

Si vous avez des questions sur l'export, contactez-nous à contact@yukibuy.com — nous vous guidons.
